# 二维码生成网站

## 启动项目

First, run the development server:
```bash
npm install
# or
yarn install
# or
pnpm install
```

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

创建数据库表
```bash
npx  prisma migrate dev  
```
